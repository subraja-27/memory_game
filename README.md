🎮 Find The Match Game

A fun, browser-based memory card matching game built with **jQuery**, **HTML**, and **CSS**. Flip the cards, remember where the matching pairs are, and try to clear the board with the best score possible!

🔗 **Live Demo:** [subraja-27.github.io/memory_game](https://subraja-27.github.io/memory_game/)

✨ Features

- 🃏 Emoji-themed card pairs that shuffle on every game
- 🔄 Smooth card flip animations
- 🧠 Classic memory / concentration gameplay — find all the matching pairs
- 🏆 Live score tracking as you play
- 🔁 Restart button to reset the board and play again
- 📱 Simple, clean, responsive layout

🕹️ How to Play

1. Click on any card to flip it and reveal what's underneath.
2. Click a second card to try to find its match.
3. If the two cards match, they stay face-up and your score increases.
4. If they don't match, the cards flip back over — try to remember them for next time!
5. Keep going until all pairs are matched.
6. Hit **Restart Game** anytime to shuffle the board and start fresh.

🛠️ Built With

HTML5 – game structure
CSS3 – styling and flip animations
jQuery – DOM manipulation and game logic
